ASSIGNMENT 5 - Employee Registration Web Application (JSP + Servlet + JDBC)

Folder structure:
- src/assignment5/Employee.java
- src/assignment5/DBConnection.java
- src/assignment5/EmployeeRegistrationServlet.java
- web/employee-registration.jsp
- web/success.jsp
- sql/employee_table.sql

Requirements:
1. Java JDK 8+
2. Apache Tomcat (or compatible servlet container)
3. MySQL Server
4. MySQL JDBC Driver in Tomcat/lib

Setup steps:
1. Run sql/employee_table.sql in MySQL.
2. Update DB credentials in src/assignment5/DBConnection.java.
3. Deploy this web app to Tomcat.
4. Open employee-registration.jsp through Tomcat.
