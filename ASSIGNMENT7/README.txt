ASSIGNMENT 7 - Employee Registration Web Application (JSP + Servlet + JDBC)

Folder structure:
- src/assignment7/Employee.java
- src/assignment7/DBConnection.java
- src/assignment7/EmployeeRegistrationServlet.java
- web/employee-registration.jsp
- web/success.jsp
- sql/employee_table.sql

Requirements:
1. Java JDK 8+
2. Apache Tomcat (or compatible servlet container)
3. MySQL Server
4. MySQL JDBC Driver in server/lib

Setup steps:
1. Create database and table by running sql/employee_table.sql
2. Update DB credentials in src/assignment7/DBConnection.java
3. Deploy this web app to Tomcat
4. Open employee-registration.jsp in browser through Tomcat
